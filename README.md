# ChessPuzzles
Checkout the project. 
Inside **ChessPuzzles** folder start the **ChessPuzzles.xcodeproj** file.
Project should be opened in XCode.

Select simulator and run project.

# Requirements

* Minimum deployments iOS 18.1

# Architecture:
* MVVM
* SwiftUI @Observable
* Dependency Injection
* Navigation
* ScreenFactory
